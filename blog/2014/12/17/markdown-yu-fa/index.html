Markdown 语法

Dec 17th, 2014 1:32 pm

段落

整个Markdown文档由多个段落组成，段落之间使用空行分割。

  • 空行只能包含空格，制表符或空字符
  • 多个空行会被当作一个空行处理
  • 同一个段落内的多个换行符和多个空格都被当作一个空格处理
  • 建议每个段落之间做好使用明确的空行进行分割，尽管一些段落会自动换行
  • Markdown 的段落可以分为下面两大类
    • 普通段落
    • 特殊段落

普通段落

普通段落是开头不包含任何特殊标记的段落。Makrdown 会用<p>标签包裹普通段落。

特殊段落

  • 标题

  • 引用

    • 标准格式
    • 简写
    • 嵌套

  • 列表

    • 无序列表
    • 有序列表
    • 列表嵌套

  • 代码

    • 缩进四个空格或一个制表符

  • 分割线

    • 三个以上下划线
    • 三个以上星号
