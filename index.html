Dec 20th, 2014 1:42 pm

变量与字面值

字面值

内存中保存的一种数据结构，也就是具体的数值。

变量

保存字面值的内存位置，通过变量可以找到存储在该位置的数值。

  • 字面值单独出现没有任何意义（存储在内存中，又没有办法找到它）
  • 字面值可以赋给某个变量
  • 字面值可以出现在表达式中

Markdown 语法

Dec 17th, 2014 1:32 pm

段落

整个Markdown文档由多个段落组成，段落之间使用空行分割。

  • 空行只能包含空格，制表符或空字符
  • 多个空行会被当作一个空行处理
  • 同一个段落内的多个换行符和多个空格都被当作一个空格处理
