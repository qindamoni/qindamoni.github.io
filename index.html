Markdown 语法

Dec 17th, 2014 1:32 pm

段落

整个Markdown文档由多个段落组成，段落之间使用空行分割。

  • 空行只能包含空格，制表符或空字符
  • 多个空行被替换成一个空行
  • 同一个段落内的换行符，多个空格将被替换成一个空格
